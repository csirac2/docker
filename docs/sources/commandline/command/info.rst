:title: Info Command
:description: Display system-wide information.

===========================================
``info`` -- Display system-wide information
===========================================

::

    Usage: docker info

    Display system-wide information.

Example
-------

.. code-block:: bash

    $ docker info
    Containers: 10
    Images: 162
    Username: svendowideit
    Registry: https://index.docker.io/v1/
    WARNING: No swap limit support
